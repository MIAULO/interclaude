Getting Started

Quick setup process to get InterClaude running in minutes

Quick Setup Process

1. Install Dependencies

pip install mcp pyyaml

2. Clone Repository

git clone https://github.com/yourusername/interclaude cd interclaude

3. Configure Claude Code

Add to your .claude.json configuration:

{ "mcpServers": { "interclaude": { "command": "python", "args": ["path/to/interclaude_server.py"], "env": { "INTERCLAUDE_CONFIG": "config.yaml" } } } }

4. Start Server

python interclaude_server.py

5. Send First Memo

Begin AI agent collaboration immediately!
